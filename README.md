# coffee-shop-dummy
"Caffeya Coffee Shop is a Python console program that greets users, shows a coffee menu, and takes orders with a friendly, interactive experience. Perfect for Python beginners."


Caffeya Coffee Shop – Python Mini Project

This is a simple Python console program that simulates a small coffee shop interaction. The user is welcomed, asked for their name, shown a menu, and then asked to place an order. The program responds with a friendly confirmation message.

Features

Welcomes the user by name

Displays a coffee menu

Takes user input for an order

Prints a confirmation message

How to Run

Make sure you have Python 3 installed.

Download or clone this repository.

Open the project folder in your terminal or command prompt.

Run the script using:

python coffee_shop.py
